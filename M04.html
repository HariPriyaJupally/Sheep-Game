M04 Sheep Ranch

This assignment introduces more concise code with jQuery.

The code is available at https://bitbucket.org/professorcase/m04.

Results of the unit tests are available at test/test.html.


Currently, the application gets the user's name and the size of their ranch. The app calculates square miles and estimates how many sheep they can hold on their land.


Do the following:

  1. Modify at least three (and hopefully more) lines of code to use concise jQuery.
  2. Change the application name displayed in the menu to a suitable name for your application.
  3. DRY (don't repeat yourself) - move the shared code from first and last name to a new function called cleanInput that takes one argument, the original answer provided by the user.
  4. Optional: Add another button. Call a webservice when the button is clicked and display the result.

Click the button to start the app.
